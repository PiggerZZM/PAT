# PAT_A1080

考点：入门模拟、排序

难度：中等

数据：

1. 需要一个struct存学生的成绩、编号(排序后会打乱编号，但要求输出学生编号)、名次（需要根据名次来判断能否破格录取）、志愿学校编号
2. 需要一个struct存学校的招生人数、实际录取人数、录取学生编号数组

算法步骤：

1. 读入学生人数、学校个数、志愿个数，读入学校招生人数，读入学生成绩及志愿
2. 根据学生的成绩按照给定规则进行排序
3. 按同分同排名法计算学生的名次
4. 拷贝到另一个学生结构数组，按照学生序号升序排列（用于下一步按学生编号搜索学生然后对比名次）
5. 对每个学生每个志愿，按照规则录取：如果未招满则录取，如果招满，选出最后一名录取的学生，对比名次，如果相同则录取
6. 对每个学校的录取学生按编号升序排列，输出

## 注意点

第5步由于学校结构里面的录取学生数组存放的是学生的编号，只能取出最后一名录取学生的编号，但经过排序后的学生数组已经乱序，无法直接按编号取出学生的排名。

此处如果直接顺序搜索，则时间复杂度是O(kn^2)，本题时间限制为0.25s，n最大可达40000会超时.

在第4步备份一个学生数组按学号升序排列，则第5步可直接取出学生的排名，时间复杂度为O(kn).

- 算法笔记上的方法：在学校结构里增设一个``last_student``变量用来记录最后一个学生的排序后的编号（或者排名），这样直接避免了搜索，是更节省空间的方法。
