# PAT_A1050

考点：哈希表，ASCII码

难度: 容易

数据：用一个大小为128的哈希表统计对应ASCII码的字符是否出现过

算法步骤：

1. 读入第一行字符
2. 读入第二行字符
3. 遍历第二行的字符数组用哈希表统计对应ASCII码的字符是否出现
4. 遍历第一行的字符数组，用哈希表检查每一个字符有无在第二行出现，若无则输出，有则不输出

注意点:
1. 读取一行字符：PAT的编译器不识别``gets()``函数，需要用``getchar()``逐个字符读入，直到读到``'\n'``，然后手工加上``'\n'``和``'\0'``
2. ``char``变量实际存放的就是字符的ASCII码

拓展：

ASCII码中不可见字符是0-31和127
