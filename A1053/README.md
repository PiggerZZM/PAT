考点：树的先根遍历、DFS

难度：容易

数据：

采用静态链表的方法，结点结构为：

    struct node
    {
        int weight; // 结点的权值
        bool isleaf;    // 记录是否为叶结点
        vector<int> child;  //  邻接表
    }

结点的结构数组：``node Node[]``

记录路径的工作变量：``vector<int> ans``
